My android letters

Add it to your build.gradle with:
```gradle
allprojects {
    repositories {
        // ...
        maven { url "https://jitpack.io" }
    }
}
```
and:

```gradle
dependencies {
    compile 'com.github.keygenqt:android-letters:0.0.0'
}
```
